# 基于create-react-app产生的ant-design的项目

dev：
```
$ git clone https://github.com/catezhao1985/ant-demo.git 
$ cd ant-demo
$ yarn
$ yarn start
```

prod：
```
$ yarn build
```

  - 支持 post&get mock data
  - 支持theme自定义
  - 封装了fetch方法
